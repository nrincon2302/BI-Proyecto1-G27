## ¿Cómo Ejecutar?
Ejecutar individualmente POR PRIMERA VEZ con:
* `python -m venv venv`
* `venv\Scripts\activate`
* `pip install fastapi uvicorn`
* `uvicorn src.main:app --reload`

Ejecutar después de instalado con:
* `venv\Scripts\activate`
* `uvicorn src.main:app --reload`
